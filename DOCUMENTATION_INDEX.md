## 🛠️ Startup Scripts

### start-ollama.bat (Windows)
**Purpose:** Automated setup for Windows
**Usage:** Double-click or run in command prompt
**Does:**
1. Checks Ollama installation
2. Verifies server is running
3. Pulls Mistral model if needed
4. Builds backend
5. Runs backend

---

### start-ollama.sh (macOS/Linux)
**Purpose:** Automated setup for Unix-like systems

## 🚀 Quick Start Commands

```bash
# 1. Install Ollama
# Download from https://ollama.ai/download

# 2. Pull Mistral
ollama pull mistral

# 3. Start Ollama
ollama serve

# 4. In new terminal, start backend
cd c:\workspace\EBP\backend
java -jar target/backend-dev.jar

# 5. Test in chatbot
# Ask: "Predict leave for employee 42"
```
