# RAG Integration - Quick Reference\n\n## рҹҡҖ Quick Start (5 minutes)\n\n### 1. Start Milvus\n```bash\ndocker run -d --name milvus -p 19530:19530 -p 9091:9091 milvusdb/milvus:latest\n```\n\n### 2. Add Dependencies to pom.xml\n```xml\n<dependency>\n    <groupId>io.milvus</groupId>\n    <artifactId>milvus-sdk-java</artifactId>\n    <version>2.3.5</version>\n</dependency>\n<dependency>\n    <groupId>dev.langchain4j</groupId>\n    <artifactId>langchain4j</artifactId>\n    <version>0.27.0</version>\n</dependency>\n```\n\n### 3. Add Configuration\n```properties\nmilvus.host=localhost\nmilvus.port=19530\nembedding.model=text-embedding-3-small\nembedding.dimension=1536\nllm.api.key=${OPENAI_API_KEY}\n```\n\n### 4. Copy Service Files\n- `EmbeddingService.java` вҶ’ `backend/src/main/java/com/ebp/service/`\n- `VectorStoreService.java` вҶ’ `backend/src/main/java/com/ebp/service/`\n- `RAGService.java` вҶ’ `backend/src/main/java/com/ebp/service/`\n- `RAGResource.java` вҶ’ `backend/src/main/java/com/ebp/resource/`\n\n### 5. Add Frontend Component\n- `RAGTraining.jsx` вҶ’ `frontend/src/components/`\n- `RAGTraining.css` вҶ’ `frontend/src/components/`\n\n### 6. Update App.jsx\n```jsx\nimport RAGTraining from './components/RAGTraining'\n\n// Add route\n<Route path=\"/rag\" element={<RAGTraining />} />\n```\n\n### 7. Update Navbar.jsx\n```jsx\n<NavLink to=\"/rag\">рҹӨ– RAG Training</NavLink>\n```\n\n### 8. Run\n```bash\ncd backend && mvn clean quarkus:dev\ncd frontend && npm run dev\n```\n\n## рҹ“Ҡ API Quick Reference\n\n### Train Single Employee\n```bash\ncurl -X POST http://localhost:8080/api/rag/train \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"employeeId\": 1}'\n```\n\n### Train Batch\n```bash\ncurl -X POST http://localhost:8080/api/rag/train/batch \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"employeeIds\": [1,2,3,4,5]}'\n```\n\n### Get Prediction\n```bash\ncurl http://localhost:8080/api/rag/predict/employee/1?days=30\n```\n\n### Search Similar\n```bash\ncurl -X POST http://localhost:8080/api/rag/search \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"employeeName\": \"John\",\n    \"role\": \"Cashier\",\n    \"location\": \"Store London\",\n    \"limit\": 5\n  }'\n```\n\n### Get Stats\n```bash\ncurl http://localhost:8080/api/rag/stats\n```\n\n### Clear All\n```bash\ncurl -X DELETE http://localhost:8080/api/rag/clear\n```\n\n## рҹҸ—пёҸ
